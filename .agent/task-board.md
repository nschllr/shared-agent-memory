# Task Board

## Rules

- Sections are `Active`, `Later`, and `Done Recently`.
- Each task must use `Next`, `Plan`, and `Claim`.
- `Next` is one immediate action.
- `Plan` is `none` or one `docs/plans/<slug>.md`.
- `Claim` is `none` unless substantial or overlapping work is active.
- Keep `Done Recently` to at most 5 one-line items.
- New plan-backed tasks default to `Later` unless the user says to implement now.
